# 📊 OLA Ride Analytics Dashboard

An end-to-end Data Analytics project built using Power BI to analyze ride bookings, revenue trends, cancellations, and customer behavior.

## 🚀 Project Overview
This dashboard provides insights into ride volume, booking status, revenue by payment method, customer trends, and cancellation patterns to support data-driven decision making.

## 📈 Key Insights
- Total Bookings: 103,024  
- Successful Bookings: 63,967  
- Cancelled Bookings: 28,933  
- Total Booking Value: 35M  
- Revenue analysis by Payment Method (Cash, UPI, Cards)  
- Ride volume trends (daily & monthly)  
- Top 5 customers by booking value  
- Customer vs Driver cancellation analysis  

## 🛠 Tools Used
- Power BI Desktop  
- DAX  
- Power Query (ETL)  
- Data Modeling  

## 🎯 Business Value
- Identifies ride success & cancellation patterns  
- Tracks revenue distribution by payment type  
- Analyzes customer behavior & booking trends  
- Supports operational and strategic decisions
